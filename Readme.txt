Use of application:
To use the application one will need to create a new login to gain access to images and categories.
Then you are able to upload a picture to the site, with the option to make it public, if you make it public you will be able to see it
on the home page with all the other users pictures which are made public.

There is a picture attached to the folder which is usable as a test picture.
